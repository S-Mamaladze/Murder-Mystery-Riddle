# 🎮 Try the SQL Murder Mystery Yourself!

Think you can find the killer? This project is based on the famous SQL Murder Mystery challenge. You can run the investigation steps yourself using the official interactive terminal.

## 🚀 How to Play

Visit the Game: Go to the SQL Murder Mystery Website.

Access the Terminal: Scroll down to the interactive SQL editor at the bottom of their page.

Use My Scripts: You can use the queries found in my Exploratory_Queries.sql file to trace the evidence step-by-step.

Final Challenge: See if you can find the mastermind using my "Master Query" found in the README.md.

## 📊 Database Schema

To help you navigate, here is the structure of the database you will be querying:

crime_scene_report: Start here to find the date and city.

person: The main directory of citizens.

interview: Where the witness testimonies are stored.

get_fit_now_member: Gym records for the suspects.

drivers_license: Physical descriptions and car details.

facebook_event_checkin: The key to finding the mastermind!

## Note: This project was migrated and tested in Microsoft SQL Server (SSMS) to demonstrate professional-grade database management and troubleshooting.
